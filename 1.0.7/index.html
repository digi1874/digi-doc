digi 1.0.7

一个简单的 js 框架

插件

  • router
  • classname
  • refs

安装

yarn add -D digi

实例

  • demo

简单例子

import digi, { createData } from 'digi'

// 创建监听数据
const data = createData()

// 添加元素
digi({ text: data.$tp('a') })

console.log(document.body.lastElementChild.outerHTML)
// => <div></div>

data.a = 321
console.log(document.body.lastElementChild.outerHTML)
// => <div>321</div>
